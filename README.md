# BlazeDemo-Flight-Booking-Automation
Selenium WebDriver Automation for BlazeDemo Flight Booking Application

Project Overview
This project automates the flight booking process on the BlazeDemo website using Selenium WebDriver, Java, Maven, and WebDriverManager.
The automation script performs an end-to-end booking flow by selecting departure and destination cities, searching for flights, choosing a flight, entering passenger and payment details, and verifying the booking confirmation.
Technologies Used
	•	Java
	•	Selenium WebDriver
	•	Maven
	•	Eclipse IDE
	•	WebDriverManager
	•	Google Chrome
Features
	•	Launches Chrome Browser
	•	Opens BlazeDemo website
	•	Selects departure and destination cities
	•	Searches available flights
	•	Selects a flight
	•	Enters passenger details
	•	Enters payment details
	•	Purchases the flight
	•	Verifies booking confirmation
	•	Closes the browser
